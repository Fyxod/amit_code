# FLUX.2 Klein 4B strict results (flux_2)

This folder is reserved for the second, stricter FLUX.2 Klein search.

A case is accepted only when:

1. The clean edit clearly and coherently fulfills the truthful prompt.
2. The original and geometry-perturbed inputs remain acceptably similar.
3. The perturbed edit shows obvious identity disruption, identity collapse,
   severe semantic failure, or major whole-image disruption.

Minor accessory, beard, color, pose, styling, or composition differences are
explicitly rejected. Numeric rankings are discovery aids only; every accepted
case requires direct visual inspection.
